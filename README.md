
# 🏋️‍♂️ AI Real-Time Gym Coach

An intelligent fitness assistant that uses **Computer Vision + AI Voice Coaching + Real-Time Pose Detection** to help users perform workouts with proper form, track progress, and stay motivated.

Built with **Python**, **Streamlit**, **OpenCV**, **MediaPipe**, **SQLite**, and **Groq LLM**.

## 🚀 Features

### 🎯 Real-Time Workout Detection
Uses webcam-based pose estimation to detect exercise movement and automatically count reps.

### 🧠 AI Voice Coach
Smart voice assistant provides:

- Motivation during workouts  
- Form correction tips  
- Set completion feedback  
- Workout completion encouragement  

Powered by Groq LLM.

### 📊 Workout Progress Tracking

Tracks:

- Total Reps  
- Current Set Reps  
- Sets Completed  
- Session Duration  

### 📝 Workout History

Stores workout sessions in SQLite database and displays history by date.

### 🔐 User Login System

Simple username-based authentication for personalized workout history.

### 🏋️ Supported Exercises

- Squats  
- Push-ups  
- Biceps Curls  
- Shoulder Press  
- Lunges

## 🎮 How It Works

1. Login with username  
2. Select exercise, sets, reps  
3. Start workout  
4. Webcam tracks movement in real-time  
5. AI Coach gives voice feedback  
6. Progress saved automatically
